# Contributing to The Masterplan

## Repo Structure

| Directory | Purpose | Who writes here |
|-----------|---------|-----------------|
| `research/` | Scout reports, market research, competitor analysis | Scout, Atlas |
| `architecture/` | System design docs, ADRs, technical specs | Atlas, Rex, Forge |
| `playbooks/` | Runbooks, SOPs, agent workflows, cron configs | Kev, Dot, any agent |
| `tools/` | Scripts, utilities, and factory automation tooling | Rex, Forge |

## Conventions

- All docs in Markdown.
- Name files descriptively: `research/2026-02-08-cerebras-evaluation.md`
- PRDs go in `architecture/`.
- Keep the README as a high-level summary; detailed docs go in subdirectories.
